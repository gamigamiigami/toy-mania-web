# 画像アセットの置き場所

このフォルダに次の2ファイルを置くと、Farm ステージに反映されます。

- `background.png` … 牧場の背景画像（的の円盤は含めない）
- `targets.png` … 的のスプライトシート（3×3＝9アイコンを格子状に並べた1枚）

## 仕様
- `targets.png` は 3列×3行 の格子。各セルが1アイコン。
  - インデックスは左上から右へ 0,1,2 / 3,4,5 / 6,7,8。
  - 各セルは正方形が望ましい（円形に切り抜いて表示します）。
  - 背景は白でもOK（円形クリップで角は隠れます）。透過PNGならより綺麗です。
- `background.png` は 16:9 推奨（正方形でも cover 表示で自動調整）。

## 追加方法（GitHub Web）
1. リポジトリの `public/scene/` を開く
2. Add file → Upload files で上記2枚をアップロード
3. `main` ブランチへコミット → 自動デプロイで反映

ファイルが無い間は、背景はグラデーション、的は黄色い円で代替表示されます。
